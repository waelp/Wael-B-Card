# Business Card Vault - Design Document

## نظرة عامة
تطبيق موبايل احترافي لمسح بطاقات العمل باستخدام OCR وAI، مع استخراج البيانات تلقائياً، منع التكرار، البحث المتقدم، والمزامنة التلقائية مع Google Sheets. يدعم اللغتين العربية والإنجليزية مع واجهة عصرية وتأثيرات حركية ديناميكية.

## قائمة الشاشات

### 1. Home Screen (الشاشة الرئيسية)
**المحتوى الأساسي:**
- قائمة بطاقات العمل المحفوظة (FlatList)
- عرض البطاقات في شكل Cards مع:
  - اسم الشركة (Company Name)
  - الاسم الكامل (Full Name)
  - المسمى الوظيفي (Job Title)
  - القسم (Department)
  - رقم الموبايل (Mobile)
  - البريد الإلكتروني (Email)
- زر عائم (FAB) لإضافة بطاقة جديدة
- شريط بحث في الأعلى

**الوظائف:**
- الضغط على رقم الهاتف → اتصال مباشر
- الضغط على البريد الإلكتروني → فتح تطبيق البريد
- الضغط على البطاقة → فتح صفحة التفاصيل
- سحب البطاقة لليسار → خيارات (حذف، تعديل)

### 2. Scan Screen (شاشة المسح)
**المحتوى الأساسي:**
- كاميرا لالتقاط صورة البطاقة
- إطار توجيهي لوضع البطاقة
- زر التقاط الصورة
- معاينة الصورة الملتقطة
- نموذج استخراج البيانات مع الحقول:
  - Company Name
  - Full Name
  - First Name
  - Last Name
  - Job Title
  - Department
  - Mobile Number
  - Phone Number
  - Email Address
  - Date (تلقائي)

**الوظائف:**
- التقاط صورة البطاقة
- استخراج البيانات تلقائياً باستخدام OCR + AI
- تعديل البيانات المستخرجة يدوياً
- التحقق من عدم وجود تكرار (Mobile Number)
- حفظ البطاقة محلياً

### 3. Card Detail Screen (شاشة تفاصيل البطاقة)
**المحتوى الأساسي:**
- صورة البطاقة (إن وجدت)
- جميع بيانات البطاقة المحفوظة
- أزرار إجراءات سريعة:
  - اتصال
  - إرسال بريد إلكتروني
  - حفظ في جهات الاتصال
  - مشاركة

**الوظائف:**
- عرض جميع التفاصيل
- تعديل البيانات
- حذف البطاقة
- مشاركة معلومات الاتصال

### 4. Search Screen (شاشة البحث)
**المحتوى الأساسي:**
- حقل بحث
- فلاتر (حسب الشركة، القسم، المسمى الوظيفي)
- نتائج البحث الفورية

**الوظائف:**
- بحث فوري أثناء الكتابة
- بحث بدون حساسية لحالة الأحرف
- دعم البحث الجزئي
- فلترة حسب معايير متعددة

### 5. Settings Screen (شاشة الإعدادات)
**المحتوى الأساسي:**
- تبديل اللغة (عربي/إنجليزي)
- إعدادات المزامنة مع Google Sheets
- إعدادات الإشعارات
- خيارات التصدير (CSV)
- معلومات التطبيق

**الوظائف:**
- تغيير اللغة
- تفعيل/تعطيل المزامنة التلقائية
- تصدير البيانات

## مسارات المستخدم الرئيسية

### 1. مسح بطاقة جديدة
1. المستخدم يضغط على زر FAB في الشاشة الرئيسية
2. يفتح الكاميرا → يلتقط صورة البطاقة
3. النظام يستخرج البيانات تلقائياً
4. المستخدم يراجع ويعدل البيانات إذا لزم الأمر
5. يضغط على "حفظ"
6. النظام يتحقق من عدم وجود تكرار
7. يحفظ البطاقة ويعود للشاشة الرئيسية

### 2. البحث عن بطاقة
1. المستخدم يضغط على حقل البحث في الشاشة الرئيسية
2. يكتب اسم الشركة أو الشخص أو أي معلومة
3. النتائج تظهر فورياً أثناء الكتابة
4. يضغط على البطاقة المطلوبة → تفتح صفحة التفاصيل

### 3. الاتصال بشخص
1. من الشاشة الرئيسية أو صفحة التفاصيل
2. المستخدم يضغط على رقم الهاتف
3. يفتح تطبيق الهاتف مباشرة مع الرقم

## مبادئ التصميم

### 1. iOS-First Design
- اتباع Apple Human Interface Guidelines
- استخدام SF Symbols للأيقونات
- تصميم يشبه تطبيقات iOS الأصلية
- دعم Dark Mode تلقائياً

### 2. Mobile Portrait Orientation (9:16)
- جميع الشاشات مصممة للاستخدام بيد واحدة
- العناصر المهمة في متناول الإبهام
- الأزرار الرئيسية في الأسفل

### 3. Modern & Professional
- تصميم نظيف وبسيط
- مساحات بيضاء كافية
- تباين واضح بين العناصر
- خطوط واضحة وسهلة القراءة

### 4. Animated & Dynamic
- انتقالات سلسة بين الشاشات
- تأثيرات حركية عند التفاعل
- Haptic feedback عند الضغط
- Loading states واضحة

## التأثيرات الحركية

### 1. Card Animations
- Fade in عند التحميل
- Scale animation عند الضغط (0.97)
- Slide animation عند الحذف
- Shimmer effect أثناء التحميل

### 2. Screen Transitions
- Slide from right للشاشات الجديدة
- Fade transition للـ modals
- Scale + fade للـ popups

### 3. Button Interactions
- Scale down (0.97) عند الضغط
- Haptic feedback (Light)
- Color transition (200ms)

### 4. List Animations
- Staggered fade in للعناصر
- Smooth scroll
- Pull to refresh animation

## الميزات التقنية

### Local Storage
- استخدام AsyncStorage لحفظ البيانات محلياً
- البيانات تبقى حتى بعد إغلاق التطبيق
- لا حاجة لإنشاء حساب مستخدم

### OCR & AI
- استخدام AI من الخادم المدمج لاستخراج البيانات
- دعم بطاقات العمل بالعربية والإنجليزية
- تطبيع أرقام الهواتف للصيغة الدولية

### Duplicate Prevention
- استخدام Mobile Number كمفتاح فريد
- التحقق قبل الحفظ
- إظهار تنبيه إذا كانت البطاقة موجودة

### Google Sheets Integration (Optional)
- مزامنة تلقائية يومية
- إنشاء ورقة "سجل العملاء" تلقائياً
- منع التكرار على مستوى Google Sheets

## ملاحظات التطوير

1. **لا حاجة لطلب أذونات جهات الاتصال** - البيانات تبقى داخل التطبيق فقط
2. **استخدام AI المدمج في الخادم** - لا حاجة لـ API keys خارجية
3. **التركيز على الوظائف أولاً** - ثم التأثيرات الحركية
4. **اختبار شامل** - جميع الأزرار يجب أن تعمل
5. **دعم RTL كامل** - للغة العربية
